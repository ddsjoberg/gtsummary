modify_column_hide {gtsummary}R Documentation

Modify Hidden Columns

Description

Use these functions to hide or unhide columns in a gtsummary tables.

Usage

modify_column_hide(x, columns)

modify_column_unhide(x, columns)

Arguments

x

gtsummary object

columns

vector or selector of columns in x$table_body

Example Output

Example 1

See Also

Other Advanced modifiers: modify_fmt_fun(), modify_table_body(), modify_table_styling()

Examples

# Example 1 ----------------------------------
# hide 95% CI, and replace with standard error
modify_column_hide_ex1 <-
  lm(age ~ marker + grade, trial) %>%
  tbl_regression() %>%
  modify_column_hide(columns = ci) %>%
  modify_column_unhide(columns = std.error)
