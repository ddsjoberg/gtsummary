tab_style_bold_p.tbl_regression {gtsummary}R Documentation

Bold significant p-values

Description

Bold p-values below a chosen threshold (e.g. <.05) in tables created by tbl_regression.

Usage

## S3 method for class 'tbl_regression'
tab_style_bold_p(x, t = 0.05, ...)

Arguments

x

an object created using tbl_regression function

t

Determines the threshold below which p-values will be bold. Default is 0.05.

...

not used

Example Output

Author(s)

Daniel D. Sjoberg

Examples

tbl_lm_bold_p_ex <-
  glm(response ~ trt + grade, trial, family = binomial(link = "logit")) %>%
  tbl_regression(exponentiate = TRUE) %>%
  tab_style_bold_p()
