tab_style_bold_p.tbl_uvregression {gtsummary}R Documentation

Bold significant p-values

Description

Bold p-values below a chosen threshold (e.g. <.05) in tables created by tbl_uvregression

Usage

## S3 method for class 'tbl_uvregression'
tab_style_bold_p(x, t = 0.05, q = FALSE,
  ...)

Arguments

x

an object created using tbl_uvregression function

t

Determines the threshold below which p-values will be bold. Default is 0.05.

q

logical argument. When TRUE will bold the q-value column rather than the p-values

...

not used

Daniel D. Sjoberg

Examples

tbl_uvglm_bold_p_ex <-
  trial %>%
  dplyr::select(age, marker, response, grade) %>%
  tbl_uvregression(
    method = glm,
    y = response,
    method.args = list(family = binomial),
    exponentiate = TRUE
  ) %>%
  tab_style_bold_p(t = 0.25)
