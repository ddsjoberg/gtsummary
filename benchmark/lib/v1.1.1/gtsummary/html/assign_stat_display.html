assign_stat_display {gtsummary}R Documentation

Assign type of summary statistic

Description

Function that assigns default statistics to display, or if specified, assigns the user-defined statistics for display.

Usage

assign_stat_display(variable, summary_type, stat_display)

Arguments

variable

Vector of variable names

summary_type

A list that includes specified summary types

stat_display

List with up to two named elements. Names must be continuous or categorical. Can be NULL.

Value

vector of stat_display selections for each variable
