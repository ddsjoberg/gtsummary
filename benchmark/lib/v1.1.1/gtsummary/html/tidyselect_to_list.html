tidyselect_to_list {gtsummary}R Documentation

Convert tidyselect to variable list

Description

Functions takes a list of tidyselect formulas, e.g. list(starts_with("age") ~ "continuous"), and returns a named list, e.g. list(age = "continuous").

Usage

tidyselect_to_list(.data, x, .meta_data = NULL)

Arguments

.data

data with variables to select from

x

list of tidyselect formulas

.meta_data

meta data from tbl_summary. Default is NULL
