sort_p.tbl_summary {gtsummary}R Documentation

Sort variables in table by ascending p-values

Description

Sort variables in tables created by tbl_summary by ascending p-values

Usage

## S3 method for class 'tbl_summary'
sort_p(x, q = FALSE, ...)

Arguments

x

An object created using tbl_summary function

q

Logical argument. When TRUE will sort by the q-value column rather than the p-values

...

Not used

Value

A tbl_summary object

Example Output

Author(s)

Karissa Whiting

Examples

tbl_sum_sort_p_ex <-
  trial %>%
  dplyr::select(age, grade, response, trt) %>%
  tbl_summary(by = trt) %>%
  add_p() %>%
  sort_p()
