style_pvalue {gtsummary}R Documentation

Style p-values to be displayed in tables or text

Description

Style p-values to be displayed in tables or text

Usage

style_pvalue(x, digits = 1, prepend_p = FALSE)

Arguments

x

Numeric vector of p-values.

digits

Number of digits large p-values are rounded. Must be 1 or 2. Default is 1.

prepend_p

Logical. Should 'p=' be prepended to formatted p-value. Default is FALSE

Value

A character vector of styled p-values

Author(s)

Daniel D. Sjoberg

Examples

pvals <- c(
  1.5, 1, 0.999, 0.5, 0.25, 0.2, 0.197, 0.12, 0.10, 0.0999, 0.06,
  0.03, 0.002, 0.001, 0.00099, 0.0002, 0.00002, -1
)
style_pvalue(pvals)
style_pvalue(pvals, digits = 2, prepend_p = TRUE)
