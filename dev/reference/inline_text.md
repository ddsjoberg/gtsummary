# Report statistics from gtsummary tables inline

Report statistics from gtsummary tables inline

## Usage

``` r
inline_text(x, ...)
```

## Arguments

- x:

  (`gtsummary`)  
  Object created from a gtsummary function

- ...:

  Additional arguments passed to other methods.

## Value

A string reporting results from a gtsummary table
