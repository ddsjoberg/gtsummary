use_coverage {usethis}R Documentation

Test coverage

Description

use_coverage() Adds test coverage reports to a package that is already using Travis CI.

Usage

use_coverage(type = c("codecov", "coveralls"))

use_covr_ignore(files)

Arguments

type

Which web service to use for test reporting. Currently supports Codecov and Coveralls.

files

Character vector of file globs.
