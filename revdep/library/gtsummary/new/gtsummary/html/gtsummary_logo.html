gtsummary_logo {gtsummary}R Documentation

The gtsummary logo, using ASCII or Unicode characters

Description

Use crayon::strip_style() to get rid of the colors.

Usage

gtsummary_logo(unicode = l10n_info()$`UTF-8`)

Arguments

unicode

Whether to use Unicode symbols. Default is TRUE on UTF-8 platforms.

Examples

gtsummary_logo()
